## Tech
- **Fetching Data**
  - [Spotify Web API](https://developer.spotify.com/documentation/web-api/)

- **Data Analysis/Visualization**
  - [numpy](https://numpy.org/)
  - [WordCloud](https://amueller.github.io/word_cloud/)
  - [Matplotlib](https://matplotlib.org/)
